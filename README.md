# 2. Installeer dependencies
cd Little-Oummah-Webshop

# 3. Voeg je Claude API key toe aan GitHub Secrets:

## 💡 Hoe Vlaamse Klanten Vinden?

1. **Facebook Groepen** — "Ondernemers Antwerpen / Gent / Brugge"
2. **LinkedIn** — zoek op "zaakvoerder" + stad
3. **Google Maps** — zoek lokale bedrijven zonder goede website
4. **Lokale netwerkevenementen** — VOKA, Unizo events
